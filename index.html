##form表单

作用 是用来提交数据的 属性 一个是 action 提交的位置 # 代表的是当前页面 method 是提交方式
get 提交试讲提交的内容拼接在地址栏的后面 不安全
post 提交方式是封装起来再提交 相对安全

  • type属性
    • text 默认的 文本 的意思 name 和value 提交的时候使用
    • password 密码
    • checkbox 复选框 配合name一起使用一组控件
    • date 日期
    • file 文件
    • image 图片 src 设置图片的路径
    • submit 提交 按钮 没有name属性 value代表的是按钮的名称呢过
    • reset 同上
    • hidden 隐藏 用户看不到 但是提交的时候是可以提交上去的
  • 其他属性
    *name 提交到服务器的key
    value            如果value属性没给值 默认是空  提交到服务器的值 但可修改
size            默认是20 尺寸
maxlength        可输入最大长度
readonly        只读        不允许修改  但可以随着表单而提交
disabled        不可用    不允许修改 也不能跟随表单提交
checked            checked="checked" 单选按钮或复选框的默认选中
                畸形写法checked=""  checke
    ##select标签
    和option 一起使用的 option中的属性是value
    属性 一个是name 提交的时候使用 相当于key size 大小 代表的是下拉框选中的个数
    multiple下拉框多选 默认没有
    selected 表示默认被选中的
    ##css的引入
    目的 是引入css 对html进行装饰页面美观 还有就是是css和html结构分离 提高代码的复用型
    css引入的优先级 问题 距离标签的距离越近 优先级 就越高

##选择器的分类 还有权重
选择器是用来选择标签的 三种方式 一个是标签选择器 一个是类选择器 一个是id选择器
id优先级大于类选择器 大于标签选择器 层级选择器
float 浮动 clear 清除浮动 both清除两边的浮动

##盒子模型

  • 三个属性
    • border 盒子的边框
    • padding 盒子内部的内容 与盒子之间的距离
    • marginn 盒子与盒子之间的距离
